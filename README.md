# atividade1-segundobimestre

1- Quando o programa é executado a seguir o programa realiza o código apresentado "Olá Mundo".

2- Quando executado o programa realiza, o código apresentado escrito em uma única frase " Olá Mundo como vai".

3- O programa executa o código escrito só que separadamente, "Olá Mundo" na linha de cima e "Como vai' na linha de baixo, por conta do adicional "\n" no código.

4- Ao executar o programa, os código "cadeia", "leia", "escreva", vão fazer com que quando executado o programa salve a váriavel nome, o leia pergunta o nome que eu gostaria que aparecesse no programa, e por fim o "escreva" executa o nome. 

5- Assim que os códigos foram executados houve um somatório ocorrido pelos códigos "escreva" com parênteses do tipo "(100 + 200)" que assim quando executado o programa, será apresentado a resposta da conta. Temos também o "\n" que serve para colocar os textos sempre na linha de baixo.

6- O programa tem como função identificar se eu tenho mais de 18 anos, se a minha idade for maior ou igual a 18 anos ele vai escrever "maior de idade", com uso dos códigos "inteiro", "se", e "escreva".
